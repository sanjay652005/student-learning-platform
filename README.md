🎓 Student Learning Platform (MERN)
A full-stack web application that enables students to securely upload, access, and manage academic notes and previous year question papers through an authenticated system.
  
  
  🚀 Key Features
JWT-based authentication with protected routes
Role-based access control
Secure upload & download of notes and question papers
CRUD operations for notes management
View & download tracking system
RESTful API architecture
Structured MongoDB schema for scalable data handling


🏗 Architecture Overview
Frontend: React (Vite)
Backend: Node.js + Express
Database: MongoDB
Authentication: JSON Web Tokens (JWT)
API Communication: REST
